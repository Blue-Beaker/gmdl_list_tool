A tool to check if the copied level id is present in GD lists.  
Run `list_checker.py [FILE]`, and when you copy an level ID, it tells you whether the level is present in every list.  
FILE parameter can be an exported '.gmd' list file, or CCLocalLevels.dat in your saves, or a folder containing multiple '.gmd' list files.  

Used [Wyliemaster's GD-Save-Decryptor](https://github.com/Wyliemaster/GD-Save-Decryptor/) to decode the save file. [LICENSE for GD-Save-Decryptor](LICENSE_gd_save_decryptor)
